# v1.1.0

This release focuses on packaging and distribution convenience rather than crawler behavior changes.

## Highlights

- Precompiled binaries are now provided for common Windows, Linux, and macOS targets.
- A compressed package containing a complete data snapshot is included for easier evaluation and archival.

## Assets

- `metacritic-harvester_windows_amd64.exe`
- `metacritic-harvester_windows_arm64.exe`
- `metacritic-harvester_linux_amd64`
- `metacritic-harvester_linux_arm64`
- `metacritic-harvester_darwin_amd64`
- `metacritic-harvester_darwin_arm64`
- `SHA256SUMS.txt`
- Complete data snapshot archive

## Notes

- No major crawler feature changes are introduced in this release.
- Windows release binaries are packaged with an embedded application icon for a better desktop experience.
